## 2. 测试分层策略

```
           ┌─────────────────────────┐
           │   E2E (手动 / 冒烟)      │  ← 全链路用户场景
           ├─────────────────────────┤
           │   API 集成测试           │  ← HTTP 请求/响应验证
           ├─────────────────────────┤
           │   服务层单元测试          │  ← Mock 外部依赖
           ├─────────────────────────┤
           │   工具单元测试            │  ← 每个 @Tool 独立验证
           └─────────────────────────┘
```

**测试优先级定义:**
- **P0 (阻塞发布):** 认证、核心 Agent 执行、数据持久化、安全边界
- **P1 (高优先级):** 工具调用正确性、流式接口、知识库搜索
- **P2 (中优先级):** 评测管线、审计日志、历史查询
- **P3 (低优先级):** 边缘 UI 交互、非关键日志格式

## 附录 A: 快速冒烟测试脚本

```bash
#!/bin/bash
# 冒烟测试 — 30 秒快速验证核心功能
BASE="http://localhost:8081"

# 1. Health check
echo "=== Health ==="
curl -s $BASE/actuator/health | grep -q "UP" && echo "PASS" || echo "FAIL"

# 2. Register
echo "=== Register ==="
REG=$(curl -s -X POST $BASE/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username":"smoke","password":"smoke123"}')
TOKEN=$(echo $REG | grep -o '"token":"[^"]*"' | head -1 | sed 's/"token":"//;s/"//')
[[ -n "$TOKEN" ]] && echo "PASS" || echo "FAIL"

# 3. Agent Run
echo "=== Agent Run ==="
AGENT=$(curl -s -X POST $BASE/api/agents/run \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  --data-raw '{"taskDescription":"What time is it"}')
echo $AGENT | grep -q "COMPLETED" && echo "PASS" || echo "FAIL"

# 4. Agent Runs List
echo "=== Runs List ==="
curl -s -H "Authorization: Bearer $TOKEN" $BASE/api/agents/runs | grep -q "success" && echo "PASS" || echo "FAIL"

# 5. SSE Stream
echo "=== SSE Stream ==="
SSE=$(curl -s -N -X POST $BASE/api/agents/run/stream \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  --data-raw '{"taskDescription":"Check platform status"}' 2>&1)
echo $SSE | grep -q "event:done" && echo "PASS" || echo "FAIL"

echo "=== Smoke Test Complete ==="
```
