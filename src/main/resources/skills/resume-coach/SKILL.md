# Resume Coach Skill

## 适用场景
用户需要简历评估、修改建议或求职材料优化。

## 输入信息
- 简历文档 ID (documentId) — 已上传到知识库的简历文件
- 或用户直接粘贴的简历文本
- 可选的职业目标/目标岗位信息

## 执行步骤
1. 获取简历内容（document.extract_text 或用户直接输入）
2. 获取用户职业背景信息（memory.recall）
3. 多维度分析简历：
   - 结构与排版
   - 工作经历描述（STAR 法则）
   - 技能展示与关键词
   - 教育/项目经历
   - 数据化与量化成果
   - 岗位匹配度
4. 搜索相关知识库补充行业标准
5. 生成结构化改进建议
6. 保存分析报告（note.create）

## 输出格式
```
## 简历分析报告

### 总体评分: XX/100

### 各维度评估
1. 结构与排版: X/10
   [具体建议]
2. 工作经历描述: X/10
   [具体建议]
3. 技能展示: X/10
   [具体建议]
...

### TOP 3 优先改进
1. **[最重要改进项]**
   - 问题: ...
   - 建议: ...
   - 示例（修改前/后）: ...

2. ...

3. ...

### 后续行动建议
- 建议定期更新简历
- 针对不同岗位定制版本
```

## 注意事项
- 保持建议具体、可操作、非模板化
- 语气积极鼓励，不批评用户真实经历
- 不建议用户编造或夸大经历
- 尊重用户隐私
- 同一份简历针对不同目标岗位可能有不同建议
