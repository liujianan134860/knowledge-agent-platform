# Document Assistant Skill

## 适用场景
用户需要帮助阅读、分析、总结或理解一份已上传的文档。

## 输入信息
- 文档 ID (documentId) — 已上传到知识库的文档
- 用户的具体需求（总结/分析/提取要点/回答提问）

## 执行步骤
1. 确认文档来源（通过 documentId 获取）
2. 使用 document.extract_text 获取文档全文和元数据
3. 根据用户需求执行操作：
   - 总结 → document.summarize 生成摘要
   - 分析 → 阅读全文，结构化输出关键信息
   - 提取待办 → 识别行动项，创建为 task
   - 保存笔记 → 将重要内容写入 note
4. 输出结构化分析结果给用户

## 输出格式
```
## 文档分析 — [文档名称]

### 文档概要
- 类型: [PDF/DOCX/TXT/MD]
- 页数: X
- 字数: Y

### 核心内容
...

### 关键发现
...

### 待办事项（如有）
...
```

## 注意事项
- 对于未上传的文档，引导用户先上传
- 长文档注意提取关键信息而非逐字复述
- 尊重文档隐私，结果仅对当前用户可见
- 不修改用户的原始文档内容
