# Research Skill

## 适用场景
用户需要搜索互联网信息、进行深度调研或了解某个主题的最新动态。

## 输入信息
- 调研主题/问题（用户消息）
- 可选的搜索源偏好（web / wikipedia）

## 执行步骤
1. 理解用户调研主题，明确调研目标
2. web.search(source=web) 进行互联网搜索
3. web.search(source=wikipedia) 补充百科资料（如适用）
4. knowledge.search 搜索本地知识库
5. 综合分析：提取核心观点、交叉验证、客观结论
6. note.create 持久化调研报告

## 输出格式
```
## 调研报告：[主题]

### 背景
[调研背景和目的]

### 关键发现
1. [发现1] — 来源: [URL]
2. [发现2] — 来源: [URL]
...

### 结论
[综合分析后的结论]

### 参考来源
- [来源1](URL)
- [来源2](URL)
```

## 注意事项
- 多源交叉验证，区分事实和观点
- 信息不足时诚实告知
- 引用来源包含 URL
