# Daily Planning Skill

## 适用场景
用户需要规划今天要做的事，或者想知道今天该做什么。

## 输入信息
- 用户的任务描述（可能包含今天的目标或关注领域）
- 系统当前时间
- 用户现有待办任务
- 用户偏好和习惯

## 执行步骤
1. 获取当前日期和时间（datetime.now）
2. 查询现有任务列表（task.list）
3. 查询用户偏好（memory.recall，key 如 work_hours, focus_areas, energy_pattern）
4. 分析用户目标和现有任务
5. 创建必要的任务（task.create，设置合理优先级）
6. 设置关键提醒（reminder.create）
7. 输出结构化的今日计划

## 输出格式
```
## 今日计划 — 2026年X月X日 星期X

### ⭐ 优先完成
1. [任务名] (HIGH) — 建议时段
2. [任务名] (MEDIUM)

### 📋 其他任务
...

### ⏰ 关键提醒
- 时间: 提醒内容

### 💡 时间分配建议
...
```

## 注意事项
- 不要创建重复的任务
- 为每个任务设置合理的优先级
- 考虑用户的偏好和工作习惯
- 计划要可行，不要过载
